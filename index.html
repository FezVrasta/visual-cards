Visual Cards

Examples

Examples given below are added to demonstrate simple usages of visual cards.
More complex applications can be written using these simple cards.
Rendering backend is selectable for these examples (DOM, Canvas or WebGL).
  • Example 1 - Card and text rendering
  • Example 2 - Animation support
  • Example 3 - Image support

Benchmark Tests

Benchmark tests are used for measuring device's performance (eg. smart phone or TV)
Before running these examples make sure to extract the zip file ("benchmark_tests/assets/images.zip")
  • Colored Boxes Benchmark - Adds cards to application until framerate drops under a certain limit
  • Image Boxes Benchmark - Adds the card with the same image to application until framerate drops under a certain limit
  • Different Images Benchmark - Loads different images (2000 256x256 pngs) to measure memory usage
